< p >< /p >

Di HTML, elemen < p > digunakan untuk membuat paragraf teks. Singkatan "p" pada elemen < p > mengacu pada kata "paragraph" dalam bahasa Inggris, yang berarti paragraf. Elemen ini digunakan untuk mengelompokkan teks menjadi paragraf terpisah. Anda dapat menggunakan elemen < p > sebagai wadah untuk teks yang membentuk paragraf tunggal. Ketika browser mengurai dan merender kode HTML, setiap elemen < p > akan ditempatkan dalam blok terpisah, dengan baris kosong di atas dan di bawahnya untuk memberikan jarak visual antara paragraf.		 < p >Ini adalah contoh paragraf pertama.< /p >

Heading HTML

Heading merupakan element atau tag HTML yang berfungsi untuk menunjukkan bagian penting pada halaman web. element tag heading ini memiliki enam tingkatan tyang berurutan yaitu < h1 >,< h2 >,< h3 >,< h4 >,< h5 >,< h6 > yang bisa digunakan untuk menambah ke struktur halaman web. perbedaan masing-masing heading yaitu besar hingga kecil teks. contohnya tag/element < h1 > lebih besar dari element/tag < h2 > dan seterusnya.		 < h1>Ini Judul Level 1< /h1>
